# Rock, Paper, Scissors (CLI)

Play against the computer.

## How to Run

- **Prerequisites**: Python 3.8+
- **Run**:
  - Windows PowerShell:
    - `python .\main.py`
  - macOS/Linux:
    - `python3 ./main.py`

## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choices result in a draw
