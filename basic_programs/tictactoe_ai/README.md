# Tic Tac Toe (AI - Minimax)

Play tic tac toe against an AI using minimax with alpha-beta pruning.

## How to Run

- **Prerequisites**: Python 3.8+
- **Run**:
  - Windows PowerShell:
    - `python .\main.py`
  - macOS/Linux:
    - `python3 ./main.py`
